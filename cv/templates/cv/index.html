Work Experience

Marea Sistemi - Python Developer

Aggiornare e personalizzare il CRM TAImpresa, effettuandone il porting da Python 2.7 a Python 3.8 oltre che customizzarlo in base alle richieste dei clienti, sviluppandone anche la parte front-end attraverso l'utilizzo della libreria wxPython.
